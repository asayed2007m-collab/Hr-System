​👥 HR Management System (C++ & Qt)
​A professional Human Resources management application that demonstrates a deep understanding of Object-Oriented Programming (OOP) and GUI development.
​🧠 Advanced OOP Implementation
​This project is built upon the four pillars of OOP to ensure clean, maintainable, and reusable code:
​Encapsulation: All applicant data (Name, Age, Job, CV) is protected within the Applicant class using private access modifiers and exposed via public getters and setters.
​Inheritance: * Qt Framework Inheritance: The core UI logic inherits from QMainWindow, allowing the reuse of complex window management features.
​System Architecture: Designed with a modular structure where specific UI components inherit properties from base Qt widgets.
​Abstraction: Simplifies complex operations (like adding to a list or switching views) by hiding the implementation details and providing a clear interface for the user.
​Dynamic Polymorphism: Utilized through Qt's signal and slot mechanism, allowing different parts of the system to respond uniquely to user interactions.
​🚀 Key Modules
​Recruitment Pipeline: Add candidates with their specific roles and CV files.
​Decision Engine: A robust logic to Accept or Reject candidates, with real-time data migration between dynamic lists.
​HR Analytics: A dashboard that tracks total applications, acceptance, and rejection counts dynamically without data loss.

​Language: C++ (OOP Focus)
​Framework: Qt Widgets
